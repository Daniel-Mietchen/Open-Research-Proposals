Open-Research-Proposals by Daniel-Mietchen

A place to draft grant proposals in the open

Install

Forking The preferred way to make contributions is to:

  • fork the main repository
  • create a temporary development branch and make your changes there
  • commit your changes in one atomic commit per style in the master branch
  • issue a pull request to the main repository

Small edits can also be made via the github.com interface (see https://github.com/blog/844-forking-with-the-edit-button).
